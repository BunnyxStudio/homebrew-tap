# BunnyxStudio Tap

## Available formulae

- `adc`: Read-only CLI for querying App Store Connect sales, finance, reviews, and analytics data

## Install

```bash
brew tap bunnyxstudio/tap
brew install adc
```

Or:

```bash
brew install bunnyxstudio/tap/adc
```

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "bunnyxstudio/tap"
brew "adc"
```
